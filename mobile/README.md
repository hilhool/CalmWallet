# CalmWallet Mobile

Мобильный клиент CalmWallet — личный дневник трат, снижающий финансовую тревогу.
Expo (React Native) + Reanimated 3. Работает с существующим бэкендом без изменений.

## Запуск

```bash
cd mobile
cp .env.example .env   # указать LAN-IP машины с бэкендом (localhost с телефона не виден)
npm install
npx expo start         # QR-код для Expo Go, либо a / i для эмулятора
```

## Дизайн-система

### Принцип
Это личный дневник с финансовым слоем, не банк. Каждое визуальное решение
снижает напряжение: ни одного холодного оттенка, ни одной резкой анимации,
ни одного «алертного» красного.

### Типографика
- **Instrument Serif** — крупные цифры и заголовки. Editorial-характер, тепло, человечность.
- **Hanken Grotesk** — весь остальной текст. Гуманистический гротеск с мягкой геометрией.

### Цвет — «тёплая бумага» (`src/theme/index.js`)
- Фон `#F5F0E6` — бумага дневника, не белый экран.
- Текст `#322E27` — тёплые чернила, не чёрный.
- Позитивный акцент — мох `#5F7D58`; тревожный — тёплая глина `#B9694A` (не красный).
- 6 эмоций — приглушённый спектр от шалфея (покой) до глины (напряжение);
  цвет несёт эмоциональный контекст вместо текста.

### Моушн (`src/theme/motion.js`)
Один анимационный язык — дыхание: hero-зона и ожидания «дышат» в ритме
спокойного человека (~4 с цикл), всё нажимаемое мягко поддаётся под пальцем
(пружины без звона), контент «оседает», а не въезжает. Хэптика только там,
где действие имеет вес: клавиша суммы (light), выбор эмоции (medium),
сохранение (success).

### Ключевые паттерны
- **Главный экран** — hero реагирует на доминирующую эмоцию дня (градиент + дышащее пятно),
  сумма «досчитывается», AI-инсайт недели подан как цитата с цветной линейкой, не виджет.
- **Добавление траты** — одно непрерывное действие: своя клавиатура суммы → выбор эмоции
  как физический жест (выбранная подаётся вперёд) → категория → опциональный триггер/заметка.
  Каждый следующий шаг появляется после завершения предыдущего.
- **Ответ коуча** — момент присутствия: пауза с дышащей точкой, затем serif-текст
  на фоне тона выбранной эмоции. Минимум UI.
- **Аналитика** — один график-история (сглаженная линия, точки в цвет эмоции дня,
  без осей), инсайты с крупным ключевым числом, распределение эмоций одной полосой.
- **Навигация** — два знака и жест добавления, без подписей.

## Структура
